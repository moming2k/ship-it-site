Privacy Policy

Last updated: July 19, 2026

1. Short version

GitResolve doesn't have accounts, doesn't run analytics, and doesn't have a database. There is nothing of yours stored on our servers because there is no server-side storage in this product at all.

2. What actually happens when you use this site

The main page (index.html) takes a GitHub or GitLab link you paste in and opens Vercel's own project-import flow with that link pre-filled. Nothing you type is sent anywhere except as part of that redirect URL.

The upload flow (deploy-artifact.html) does more, and here's exactly what happens with each piece of data:

  • The file you upload is read entirely in your browser and sent directly to GitHub's API to create a repository, using a token obtained via GitHub's own OAuth flow. It never passes through, or is stored on, any server we control.
  • Your GitHub access token is held only in your browser tab's memory for the duration of that tab's session. It is never written to a cookie, localStorage, sessionStorage, or any server-side log, and it's discarded the moment you close or reload the tab.
  • A small relay service (a Cloudflare Worker we run) exists only to complete the one step GitHub's OAuth flow requires that a browser can't do directly — exchanging an authorization code for a token. It processes that request and returns the result; it does not log, store, or retain anything about the request.
  • Vercel and GitHub each have their own privacy policies covering what happens once you're using their services directly, which this policy doesn't cover.

3. Cookies and tracking

This site sets no cookies and includes no analytics, advertising, or tracking scripts of any kind.

4. Hosting

The site itself is static and served via GitHub Pages on a custom domain. GitHub, as the host, may process standard web server logs (e.g. IP address, user agent) as part of serving any website — see GitHub's own privacy statement for what that entails.

5. Changes to this policy

This policy may be updated from time to time. Continued use of this site after a change constitutes acceptance of the updated policy.
